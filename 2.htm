Instalando JAVA N-IDE

Para programar o jogo pelo celular a gente vai escrever codigo java e compilar.

A linguagem java é muito interessante.
Ela permite que o desenvolvedor acesse o hardware de maneiras incríveis.
O hardware é incrivel.
O hardware permite que o user comum faça coisas incriveis.
O user desenvolvedor faz coisas um pouco mais incriveis.
Mas o limite do harware é muito além.
Existe muita coisa que o hardware permite mas que ainda nao foi implementada.
Ainda tem muito coelho nessa cartola.

Então a gente vai escrever um código java que vai ser o núcleo de um app.
Um app joguinho que vamos programar.

Para converter o codigo java em um app bonitinho, com ícone, instalavel (e tudo mais) é preciso utilizar um compilador.

Existem vários softwares compiladores que convertem código em app.

Nesse tutorial a gente vai usar o Java N-ide.

Mas antes vou falar um pouco sobre o Android Studio e sobre o Aide.

O Android Studio é o software padrão recomendado para desenvolvimento android.
Mas ele é um software de computador.

O Aide é um outro app muito bom que converte código em app.
Eu utilizo o Aide a cerca de 10 anos.

Todos esses tem vantagens e desvantagens.

Para seguir esse tutorial acesse a play store e procure por "Java N-ide" e instale.
