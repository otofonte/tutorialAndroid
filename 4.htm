MainActivity.java

No canto de cima da tela, no lado esquerdo, tem um icone 'menu'.
Quando o user toca nesse icone o JAVA N-IDE exibe os arquivos do projeto.

O arquivo mais importante é o MainActivity.java

Nesse capítulo vou falar sobre esse arquivo e nos proximos capítulos falo sobre outros arquivos e pastas.

O arquivo MainActivity.java contém o método onCreate.
Nesse método está a lista de instruções que o processador deve obedecer no momento em que o app é carregado.

Por enquanto, no método onCreate está uma instrução de exibir na tela o layout que contém o texo "hello world".
Mas a gente vai mudar isso.
