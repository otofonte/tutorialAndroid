Hello World

O JAVA N-IDE permite que o user compile codigo java para aplicativos.
O JAVA N-IDE permite que o user compile codigo java para outros fins (sem ser aplicativo).

Existe a opção "novo projeto android"
Existe a opcao "novo projeto sem-ser-android"

Quando o JAVA N-IDE é iniciado pela primeira vez ele cria um projeto java sem-ser-android.

Para gerar o joguinho app a gente tem que selecionar a opção "novo projeto android"

FILE -> New -> New Android project

Repare que a tela de criar "android project" é um pouco diferente da tela "projeto sem-ser-android"

Nesse tutorial a gente faz um joguinho app android.
Um joguinho simples.
Pelo celular mesmo.
Não precisa nem ligar o computador.

Dessas opções que aparecem na hora de criar o "android project" vamos preencher com o nome "joguinho"

Quando esse projeto java-android é criado o JAVA N-IDE gera varios arquivos e pastas.
Em um outro capítulo desse tutorial eu vou explicar um-por-um desses arquivos.

Por enquanto vamos compilar isso que foi gerado automaticamente.

Depois a gente edita os arquivos (programando o jogo) e dai compila de novo.

Antes vamos simplesmente compilar o projeto em branco.
O famoso "hello world".
Se você não sabe o que é "hello world", você precisa saber. Pesquisa isso na internet.

Para compilar o projeto joguinho (hello world/em branco) é preciso tocar no icone triangulo (simbolo de play)
Esse icone fica no topo da tela.
Entre os menus.

Quando executa, o JAVA N-IDE tranforma o código java (que a gente vai editar) em um app.
E o JAVA N-IDE tenta instalar o app no celular.

Muitas vezes é preciso autorizar o JAVA N-IDE.

Nas configurações existe a opção de autorizar instalações (à partir) dessa fonte.

A fonte de instalação é o JAVA N-IDE.
Pode confiar.
Acessa as configurações e autoriza o JAVA N-IDE a instalar apps no seu celular.

Dai tem a opção de instalar o app joguinho (hello world em branco).

Assim que instala, executa o app (que vai ser o joguinho). Por enquanto não está muito interessante, mas nos próximos capítulos desse tutorial o jogo vai tomar forma.
