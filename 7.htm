Outros Arquivos Gerados

gradle é uma ferramenta que serve para compilar o código e gerar o app.

Eu programava para android antes do gradle ser obrigatório.

Eu ainda fico incomodado com esses arquivos com gradle no nome no meio do meu código.
Mas não sei de outro jeito.

Eu preciso me acostumar.

O gradle contém umas listas de instruções que o JAVA N-IDE deve obedecer para transformar o código em um app.

Nesse tutorial a gente não vai excluir nem editar nenhum arquivo desses que tem gradle no nome.

Podemos até excluir a pasta 'assets' e a pasta 'libs', mas não adianta nada porque o JAVA N-IDE cria elas de novo.
