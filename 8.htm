Vamos criar uma classe chamada ViewJoguinho
Dentro do arquivo MainActivity.java
Dentro da classe MainActivity

A classe ViewJoguinho deve ser uma extensão da classe View
public class ViewJoguinho extends View{

Para esse código funcionar é preciso importar a View
import android.view.View;

Também é preciso sobrescrever um método construtor
public ViewJoguinho(Context p){super(p);}

Para isso funcionar é preciso importar o Context
import android.content.Context;

Vamos declarar o metodo onDraw
O método onDraw desenha na tela.
Cada frame é uma execução desse método.

@Override protected void onDraw(Canvas c){
super.onDraw(c);
}

Para isso funcionar é preciso importar o Canvas
import android.graphics.Canvas;

Esse método onDraw será rodado/executado centenas de vezes.
Ele exibe todas as telas do jogo.

Por enquanto vamos apenas exibir um background azul
Paint p=new Paint();
p.setColor(Color.rgb(70,100,200));
c.drawPaint(p);

Para isso funcionar é preciso importar Paint e Color
import android.graphics.Color;
import android.graphics.Paint;

Agora vamos fazer a classe MainActivity usar essa ViewJoguinho que a gente criou.

Dentro da classe MainActivity (antes do metodo onCreate) vamos declarar um objeto vj
ViewJoguinho vj;

Dentro do método onCreate a gente inicializa o vj
vj=new ViewJoguinho(this);

Em seguida a gente altera a linha setContentView:
setContentView(vj);
Essa linha vai instruir o app a utilizar a ViewJoguinho que exibe o background azul

Depois disso a gente pode excluir a linha que importa R
A gente também pode excluir a pasta 'layouts' que está dentro da pasta 'res' (excluir o activity_main.xml que é o arquivo que contém o layout com a frase "hello world")

Antes dessas alterações o método onCreate da classe principal exibia o layout do arquivo xml
Depois dessas alterações a classe principal exibe a View que estamos criando.
