onTouchEvent

Vamos tratar o toque do user.

A princípio vamos simplesmente mudar a posicao do personagem cada vez que o user tocar a tela.

A gente cria uma variavel pulo na classe MainActivity
int pulo;

Uma boa opção é sobrecarregar o método onTouchEvent
@Override public boolean onTouchEvent(MotionEvent me){
return true;
}

A gente precisa importar o MotionEvent
import android.view.MotionEvent;

Quando o user tocar na tela o valor da variavel pula vai alternar entre 0 e 140
if(me.getAction()==MotionEvent.ACTION_DOWN){
if(pulo==0)pulo=140;
else pulo=0;
invalidate();
}

invalidate é um comando que chama a função onDraw.

MotionEvent.ACTION_UP acontece quando o user tira o dedo da tela.
MotionEvent.ACTION_DOWN acontece quando o user encosta o dedo na tela.
Existem outras variações.
E existe a possibilidade de tratar o toque independente do tipo.

Na onDraw a gente usa a variavel pulo para posicionar o personagem na tela
c.drawBitmap(personagem,(int)(medida*4),(int)(medida*13)-pulo,p);
